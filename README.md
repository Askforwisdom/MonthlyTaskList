# 月度任务清单

> **本项目由 AI 构建**

一个简约风格的桌面任务管理应用，按月份组织任务，支持完成状态跟踪和说明记录。

## 功能特性

### 任务管理
- **按月组织** - 任务按月份分类管理，支持前后月份切换
- **新增任务** - 点击 `+` 按钮快速添加任务
- **编辑任务** - 右键点击任务可编辑内容
- **删除任务** - 右键菜单支持删除操作
- **完成标记** - 勾选复选框标记任务完成
- **完成说明** - 完成任务时可添加说明，鼠标悬停查看

### 系统功能
- **开机自启** - 可设置开机自动启动
- **数据导出** - 导出任务数据备份
- **数据导入** - 导入之前备份的数据
- **本地存储** - 数据存储在应用目录，无需联网

### 界面设计
- **简约风格** - 无边框窗口，自定义标题栏
- **右侧停靠** - 窗口默认位于屏幕右侧，竖向居中
- **紧凑布局** - 竖条长方形窗口，不占用过多屏幕空间

## 快速开始

### 开发模式

```bash
# 安装依赖
npm install

# 启动应用
npm start
```

### 打包发布

```bash
npm run build
```

## 使用说明

1. **添加任务** - 点击左上角 `+` 按钮
2. **切换月份** - 点击 `<` 或 `>` 按钮
3. **完成任务** - 勾选任务前的复选框，可添加完成说明
4. **编辑/删除** - 右键点击任务
5. **设置菜单** - 点击右上角 `⋮` 按钮

## 数据存储

- 任务数据存储在应用目录的 `data/` 文件夹
- 设置数据存储在应用目录的 `settings.json`
- 首次启动自动创建存储目录
